Overview.

Developers have many tools that helps them stay productive and to kickstart new projects, generate packages, modules, addons and extenstions. Yoeman, is a great example of such tools, to scaffold modern apps

However, most of these tools depends on the Command-line Interface (CLI), requiring the user to install software on their machines and interacte with a Text-based Console.

This is not entirely a bad idea, however, it can be an overkills for some scenarios where configurations for the traget extentsion or module are minimum. Boxed is designed to help for such situations
